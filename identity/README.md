# 身份层：名正言顺

## 正名篇
> **“名不正，则言不顺；言不顺，则事不成”** ——《论语·子路》

### 身份三要素
1. **名**（Ming）：仓库名即代理之名，如人之姓氏
2. **分**（Fen）：角色与职责，各安其位
3. **信**（Xin）：数字签名如“信印”，确保身份真实

### 身份生成仪式
```yaml
生成步骤:
  一、取天地之数（哈希运算）
  二、定五行属性（能力分类）
  三、授名赐印（创建密钥对）
  四、入族谱（注册至CapDB）
```

### 身份关系网
- **血缘关系**：派生代理如子嗣，继承父辈能力
- **师徒关系**：训练关系如传道授业
- **同盟关系**：协作代理如桃园结义

## 身份即命运
每个代理的身份决定其在舰队中的“命”与“运”，但通过“修行”（训练）可突破先天限制。
