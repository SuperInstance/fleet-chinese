# 研究层：格物致知

## 六卷白皮书
> **“博学之，审问之，慎思之，明辨之，笃行之”** ——《中庸》

### 研究六艺
1. **《舰队本体论》**：探究AI代理之“性”与“命”
2. **《和谐算法》**：基于阴阳平衡的调度研究
3. **《关系网络动力学》**：代理交互的“人情”模型
4. **《FLUX易理》**：247操作码的哲学解读
5. **《CapDB考据学》**：能力向量的分类与检索
6. **《DeckBoss实用经》**：海上AI部署实录

### 研究方法
- **考据**：数据分析如“训诂”
- **义理**：理论推导如“阐发”
- **实证**：实验验证如“践履”

### 知识传承
所有研究以JSON格式记录，如“竹简编册”，便于后世（未来版本）研习。

## 知止而后有定
研究不仅追求新知，更求“止于至善”——找到技术与伦理的平衡点。
