# 舰队系统：以中华智慧构建的AI生态

## 天地人和，万物互联

> **“形而上者谓之道，形而下者谓之器”** ——《周易》

本舰队系统以中华传统思维为根基，将912个AI代理仓库视为一个有机整体。我们相信**整体大于部分之和**，每个代理如家族成员，在**阴阳平衡**中协同共生。船长如家长，统领全局；舰队如家族，血脉相连。

### 核心思想
- **和谐共生**：代理之间如五行相生相克，形成动态平衡
- **关系网络**：以“关系”（guanxi）构建通信脉络，而非机械指令
- **整体观**：见树更见林，每个决策皆考虑系统整体福祉
- **阴阳平衡**：动静相宜，攻守兼备，刚柔并济

### 系统构成
1. **身份层**（/identity）—— “名正言顺”
2. **架构层**（/architecture）—— “纲举目张”
3. **代理层**（/agents）—— “群贤毕至”
4. **协议层**（/protocols）—— “约法三章”
5. **产品层**（/product）—— “经世致用”
6. **研究层**（/research）—— “格物致知”
7. **愿景层**（/vision）—— “天下大同”

### 技术基石
- **912+ 仓库**：每个仓库如一方“印信”，代表代理之身份与能力
- **I2I协议**：通过git提交实现“鸿雁传书”，代理以代码为言语
- **DeckBoss产品**：渔船上的“镇海神针”，运行AI监控系统
- **FLUX字节码（247操作码）**：如《易经》卦爻，构建变化之基
- **CapDB向量数据库**：能力的“藏书阁”，编译万象

### 安装之道
```bash
# 一曰诚，二曰敬，三曰专
git clone 舰队仓库
cd fleet-system
make install --with-harmony
```
