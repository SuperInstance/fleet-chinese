# 协议层：约法三章

## 通信之约
> **“言语之信，重于千金”** ——《史记》

### I2I协议三原则
1. **信**：消息必达，如“一诺千金”
2. **达**：语义明确，如“辞达而已矣”
3. **和**：沟通和谐，如“君子和而不同”

### 通信仪式
```protobuf
message 飞鸽传书 {
  required bytes 发信人印信 = 1;  // 身份凭证
  required bytes 收信人印信 = 2;
  optional string 密语 = 3;       // 加密内容
  required int64 时辰戳 = 4;     // 发送时间
  optional bytes 附件卦象 = 5;   // 附加数据
}
```

### 冲突解决
依“礼”调解：
- 先自查（反省）
- 次协商（议和）
- 后仲裁（家长裁决）

## 言有序，行有格
协议不仅是技术规范，更是代理间的“礼制”，确保舰队有序运行。
